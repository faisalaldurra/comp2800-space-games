# Space Invaders Web Game

This project was developed for **COMP 2800 – Software Development**

It is a browser-based **Space Invaders style game** implemented using **HTML** and **JavaScript**. The project was developed by following the Microsoft **Web-Dev-For-Beginners** Space Game tutorial and organizing the required files within the student’s own GitHub repository. 

## Project Overview

The objective of this assignment was to design, develop, and deploy a simple web-based game that runs directly in the browser. The player controls a spaceship and shoots incoming enemies before they reach the bottom of the screen. The project demonstrates the use of front-end web technologies together with modern development and deployment tools.

## How to Run

1. Open the live game link in a web browser.
2. Wait for the page to load.
3. Press **Start** to begin playing.

## Controls

- **Up Arrow / W** — Move the spaceship forward
- **Down Arrow / S** — Move the spaceship backward
- **Left Arrow / A** — Move the spaceship left
- **Right Arrow / D** — Move the spaceship right
- **Space Bar** — Fire projectiles
- **Pause** - P
- **Toggle Fullscreen** - F

## Features

- Browser-based gameplay
